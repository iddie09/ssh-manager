SlowDNS SSH User Manager

Features:

- Create SSH users
- Set data limit (MB/GB)
- Auto delete when limit reached
- Monitor usage
- Menu panel

Install:

git clone https://github.com/iddie09/slowdns-user-manager

cd slowdns-user-manager

chmod +x install.sh

bash install.sh

Run panel:

menu
